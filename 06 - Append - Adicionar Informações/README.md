Exercício 06 – Adicionar Informações (Append)



Descrição:

Prática do recurso Append Queries para consolidar dados de diferentes tabelas com a mesma estrutura



O que foi feito:

-Importação de duas tabelas de vendas

-Uso do recurso Anexar consultas no Power Query

-Consolidação em uma única tabela



Aprendizados:

-Diferença entre Append (linhas) e Merge (colunas)

-Importância de padronizar colunas antes de anexar



Possíveis evoluções:

-Anexar várias planilhas de meses diferentes em uma única tabela histórica

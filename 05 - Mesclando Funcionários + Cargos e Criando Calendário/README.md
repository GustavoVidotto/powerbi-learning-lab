Exercício 05 – Mesclando Bases: Funcionários + Cargos

Descrição:
Exploração da função de mesclagem de consultas no Power BI, unindo informações de duas bases: funcionários e cargos

O que foi feito:
-Utilização da função “Mesclar consultas” no Power Query
-Junção da base de funcionários com a de cargos usando a coluna comum cargos
-Expansão seletiva dos campos relevantes da tabela de cargos

Aprendizados:
-Como utilizar a função Mesclar consultas no Power Query
-Importância de chaves comuns para integração de dados
-Como combinar informações de diferentes fontes para enriquecer relatórios

Possíveis evoluções:
-Criar visuais com os detalhes dos cargos por área
-Mesclar mais de duas tabelas com relacionamentos distintos
-Selecionar apenas os campos realmente necessários para performance

Exercício 02 – Importando Bases



Descrição:

Prática de diferentes formas de importação de dados no Power BI



O que foi feito:

-Importação de arquivos .csv e .xlsx

-Ajuste de delimitadores e cabeçalhos

-Padronização de nomes de colunas



Aprendizados:

-Como o Power Query interpreta diferentes fontes

-Diferença entre “Carregar” e “Transformar”



Possíveis evoluções:

-Conectar a bases externas (SQL, web, APIs)

-Comparar importação direta e conexão dinâmica

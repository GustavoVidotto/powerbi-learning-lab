Exercício 07 – Importando Pastas Inteiras / Bases Divididas



Descrição:

Importação de uma pasta contendo vários arquivos com a mesma estrutura



O que foi feito:

-Conexão com pasta do Windows

-Combinação automática dos arquivos

-Tratamento de colunas duplicadas



Aprendizados:

-Automatização da atualização de dados mensais

-Uso de parâmetros na importação



Possíveis evoluções:

-Implementar filtros para ignorar arquivos inválidos
